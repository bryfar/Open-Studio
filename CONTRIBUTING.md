# Contributing to Open Studio

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Open-Studio.git`
3. Create a branch: `git checkout -b feature/your-feature`

## Development Setup

```bash
# Install dependencies
npm install

# Run linting
npm run lint

# Start development
npm run dev
```

## Code Style

- Use TypeScript for all new code
- Follow existing code patterns
- Run `npm run lint` before committing
- Write tests for new features

## Pull Request Process

1. Update documentation if needed
2. Add tests for new functionality
3. Ensure all checks pass (`npm run lint`, `npm run test`)
4. Update the CHANGELOG if applicable
5. Request review from maintainers

## Commit Messages

Use conventional commits:
- `feat: add new feature`
- `fix: resolve bug`
- `docs: update documentation`
- `refactor: restructure code`

## Questions?

Open an issue or email: maintainers@openstudio.app
