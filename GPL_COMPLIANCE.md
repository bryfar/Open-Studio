# GPLv3 Compliance Notes

This project includes advanced-editor roadmap work inspired by GPLv3 ecosystem tools such as Shotcut:

- https://github.com/mltframework/shotcut

Compliance practices used in this repository:

1. Keep a traceable note of upstream inspiration/source references.
2. Preserve attribution in technical documentation and PR summaries.
3. Avoid copying third-party code without preserving license obligations.
4. When reusing GPL-derived logic or code in future phases, retain GPLv3-compatible distribution terms for those components.

For each advanced module added, include:

- origin reference,
- compatibility rationale,
- impacted files list.

## PR checklist (obligatorio)

1. ¿Se reutilizo codigo GPL? Si/no + fuente exacta.
2. ¿Se mantuvo atribucion visible en docs/PR?
3. ¿La distribucion del modulo afectado es GPL-compatible?
4. ¿Se listaron archivos impactados y razon tecnica?
5. ¿Se evitaron copias sin licencia o sin aviso?
